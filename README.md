# Data Structure and Algorithm Project

Simple terminal Pizza ordering system

## Used

* Linkedlist
* Arraylist
* Hashmap

## Options

* V (for view all the pizza)
* A (for add new pizza as much as needed)
* O (for order from menu)
* M (for shows the best seller)
* Q (for quit)
